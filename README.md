### MERN Flow

```
MongoDB → Express → React → Node
```

| Technology | Role |
|------------|------|
| **MongoDB** | Database for storing application data |
| **Express.js** | Backend framework for APIs |
| **React.js** | Frontend library for UI |
| **Node.js** | Runtime environment |
